# Syntax Practice Exercises

## Conditionals

1. Write a program that checks if a number is positive, negative, or zero.
2. Check if a number is even or odd.
3. Determine whether a year is a leap year.
4. Ask for a person’s age and print if they are a minor or adult.
5. Read a test score and print "Pass" if it’s ≥ 50, otherwise print "Fail".
6. Compare two numbers and print which one is larger.
7. Check if a number is divisible by 5 and 11.
8. Ask for a character and check if it’s a vowel or consonant.
9. Check if a number is within the range 1–100.
10. Given three numbers, print the largest.
11. Given three sides, check if they can form a triangle.
12. Input a character and check if it’s a digit, letter, or symbol.
13. Input a temperature and print "Cold", "Warm", or "Hot" depending on value.
14. Ask for a year of birth and determine if the person is eligible to vote (≥18 years old).
15. Check if a number is multiple of both 3 and 7.

## Loops

1. Print numbers from 1 to 10 using a for loop.
2. Print numbers from 10 down to 1 using a while loop.
3. Print all even numbers from 1 to 50.
4. Print all odd numbers from 1 to 50.
5. Compute the sum of numbers from 1 to 100.
6. Display the multiplication table for a number.
7. Calculate the factorial of a number.
8. Count the number of digits in an integer.
9. Reverse a number (e.g., 1234 → 4321).
10. Find the sum of digits of a number.
11. Print the Fibonacci series up to n terms.
12. Print all numbers between 1–100 that are divisible by both 3 and 5.
13. Ask for numbers repeatedly until the user enters 0, then print the sum.
14. Print a simple right angle triangle pattern.
15. Use nested loops to print a multiplication table (1–10).

## Methods

1. Write a method greet() that prints “Hello!”.
2. Write a method addNumbers(int a, int b) that returns the sum.
3. Create a method isEven(int n) that returns true if even.
4. Write a method maxOfTwo(int a, int b) that returns the larger.
5. Write a method square(int n) that returns the square.
6. Write a method areaOfCircle(double radius) that returns the area.
7. Write a method factorial(int n) that computes the factorial using a loop.
8. Create a method sumUpTo(int n) that returns the sum of numbers 1–n.
9. Write a method isPrime(int n) that returns true if the number is prime.
10. Create a method countVowels(String s) that returns the number of vowels.
11. Write a method reverseString(String s) that returns the reversed string.
12. Create a method isPalindrome(String s) that checks if the string is the same backward.
13. Write a method convertToFahrenheit(double celsius) that returns the temperature in Fahrenheit.
14. Write a method average(int[] numbers) that returns the average value.
15. Create a method displayPattern(int n) that prints a right triangle of stars of height n.
